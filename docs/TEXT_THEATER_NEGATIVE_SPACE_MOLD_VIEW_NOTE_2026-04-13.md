## Core Idea

Add a text-theater view mode where the scene/model is understood through **negative space** instead of positive stroke occupancy.

The practical framing is:

- ordinary text theater:
  - glyphs/strokes depict the object
  - empty space is background

- negative-space mold view:
  - the **absence** of glyph occupancy becomes the readable object volume
  - the filled field becomes the surrounding material / host block / enclosing medium
  - the viewed thing reads like a cavity, machined form, or extracted blueprint void

Reference intuition:

- not "ghost mode"
- not simple color inversion
- more like:
  - subtractive machining
  - engine block cavity shaping
  - mold/cast negative
  - absence behaving like a stable observable form

- innerlay scaffold views

- isolation of internals without the outer model fighting readability
- seeing the body/object as a carved volume instead of a decorated shell
