## The Useful Names

### 1. Output State

This is the best top-level name.

It is neutral, accurate, and operational.

It is:

- a derived surface
- a shared orienting read
- a current-board summary

It is **not**:

- a new god-object
- a replacement for truth planes
- an autonomous controller

### 2. Equilibrium

Equilibrium is not the whole machine.

Equilibrium is:

- the resting or settling truth
- the stability band
- the at-rest gauge

It matters because it gives a clean reference for deviation, recovery, and calibration.

### 3. Continuity Drill

This phrase is now real shop language.

`Run the continuity drill` means:

1. `continuity_restore`
2. theater-first live corroboration
3. classify the seam
4. continue from the last verified step

### 4. Trajectory Correlator

This is the next missing correction layer.

It should compare:

- intended sequence
- actual operations
- current shared truth

Then grade the deviation:

- `match`
- `adjacent`
- `widen`
- `drift`
- `break`

This is the "interview every op" idea in useful form.

### 5. "Any and any a"

As normal grammar, this is stylized, not standard English.  
As local doctrine, it works.

Useful local meaning:

- across any active surface
- and any adjacent or associated surface that helps corroborate the same truth

That makes it a good phrase for similarity / adjacency logic, even if it stays informal.

- continue from the last verified stitch
