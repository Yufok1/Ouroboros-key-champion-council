# Techlit Buyback Burn Potato Mode 2026-04-19

## Posture

Techlit is now treated in easy mode:

- `Pump tokenized agent` is the hourly buyback/burn furnace when armed.
- `creator fees` stay in a simple `SOL` reserve for forge work and optional burn top-ups.
- `Hold Door Raid Report` is the main artifact.
- `Plat DKP` remains an off-chain scoreboard/story/status layer.
- `camp / raid / shield / forge` remain as report categories, not payout obligations.

## What This Retires

Potato mode explicitly parks:

- holder distributions
- manual payout sessions
- settlement ledger obligations
- treasury-release ceremony
- pretending the report factory owes anybody cash

## Runtime Reading

- `technolit_distribution_packet` = buyback/burn policy surface
- `technolit_treasury_bridge_packet` = forge-and-burn reserve surface
- `holder_snapshot_packet` = scoreboard/watch posture only
- `raid_contribution_packet` = storyboard/prestige lane only
- `settlement_epoch_packet` = report clock only
- `hold_door_raid_report_packet` = primary delivery artifact
- `threat_bounty_packet` = watch/routing only unless later re-armed
