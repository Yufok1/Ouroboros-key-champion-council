# Atrai Mission And Primitives Report - 2026-04-29

## Mission Statement

Atrai is a human-governed, community-bound mobile ark faculty for civic continuity.

Its purpose is to help communities preserve life-support knowledge, mobility, shelter, energy, water, communications, archives, repair capacity, governance memory, and training under unstable conditions.

Atrai is not machine sovereignty. It is not autonomous command. It is not a static empire project.

Atrai is a mobile fortress in the civic sense: a resilient, repairable, inspectable, human-authorized body for carrying continuity when conditions change.

## Core Doctrine

Humans are the authority boundary.

AI systems may inspect, connect, summarize, simulate, propose, and record. They do not authorize themselves.

The operating rule is Source HOLD:

- visibility is not authority
- observation is not consent
- display is not control
- simulation is not deployment
- provenance records events but does not authorize them
- every real-world action remains human/community authorized

This is open surgery: the system shows the source, the cut, the diff, the receipt, the risk, and the rollback path when practical.

## Why Organisms

The organism-mainframe display treats primitives as living-looking system components because humans understand complex systems faster when components have identity, posture, role, and relationship.

The primitive is not a decorative icon on top of the organism. The primitive is the organism's body part, force path, role, and constraint.

Each organism card should expose:

- source
- primitive id
- body plan
- ports
- dependencies
- risks
- provenance receipts
- HOLD condition
- current status

This lets the display act like a visible complex-systems binder without becoming an execution authority.

## Base Body Grammar

The first stable Atrai organism grammar is:

`serpentine coil + tensile line + cubic paneling`

Meaning:

- serpentine coil = phase body, route continuity, transport spine
- tensile line = force closure, tendon, restraint, cable, sensing path
- cubic paneling = repairable shell, service cell, transfer floor, modular surface

Together they form a visual language for mobile, repairable, phase-controlled civic systems.

## No-Chance Fit Rule

A primitive connection is valid only if:

1. port classes are compatible
2. units match or a named converter exists
3. direction is explicit
4. source and sink are named
5. moving parts name charge, catch, and loss budget
6. tensile members name both anchors
7. panel members name hinge/lock/edge geometry
8. occupancy paths name safe/unsafe zones
9. AI-visible surfaces name source status
10. real-world action routes through Source HOLD

If any check fails, the display may show adjacency or speculation, but not fit.
