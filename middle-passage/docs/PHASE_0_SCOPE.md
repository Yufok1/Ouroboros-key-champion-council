# Phase 0 Scope

This package begins as a GitHub-ready scaffold, not a claim-making forensic
engine.

## What Phase 0 Does

- Loads flexible voyage CSV records.
- Builds synthetic-safe survey-priority zones.
- Exports GeoJSON FeatureCollections.
- Carries protection notices and sensitivity labels.
- Provides tests and CI.

## What Phase 0 Does Not Do

- It does not identify human remains.
- It does not publish high-precision candidate burial coordinates.
- It does not replace descendant/community governance.
- It does not replace maritime archaeology, oceanography, or legal review.

## Required Before Real Research Outputs

1. Source license validation.
2. SlaveVoyages schema validation.
3. NOAA/GEBCO/ISA source adapter review.
4. Descendant/community governance model.
5. Sensitive-coordinate release policy.
6. Expert review of descent, drift, preservation, and detectability assumptions.

## Public Language

Use:

- survey-priority zone
- memorial zone
- protected ancestral space
- source confidence
- uncertainty radius

Avoid:

- treasure map
- salvage zone
- discovery claim
- proof of remains
- cargo except when quoting historical records
