# Contributing

This project handles histories of mass death, enslavement, and ancestral memory.
Contributions must prioritize dignity, accuracy, and protection over speed.

## Rules

- Cite sources for every historical, legal, oceanographic, or forensic claim.
- Do not publish sensitive high-precision coordinates without governance review.
- Prefer `survey-priority zone` or `memorial zone` over extractive language.
- Keep uncertainty visible in outputs.
- Do not call model output proof of remains.
- Use synthetic data in tests.

## Good First Contributions

- Add source schema adapters.
- Add data license notes.
- Improve uncertainty propagation.
- Add generalized map export.
- Add legal/heritage protection metadata.
