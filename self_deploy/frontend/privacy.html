Privacy Policy

Last updated: February 26, 2026

This policy applies to the Champion Council deployment currently serving this page.

What This Service Does

Champion Council is a model orchestration runtime and demo interface. It processes API requests, tool calls, and workflow operations in order to return results to users and connected clients.

Data We Process

  • Request metadata such as timestamps, tool names, and operation durations.
  • Arguments sent to endpoints such as /api/tool/{tool_name}.
  • Tool output and error payloads needed to return responses and show activity history.

How Data Is Used

  • To execute requested operations and return results.
  • To render operational telemetry in the Activity feed.
  • To diagnose runtime errors and improve reliability.

Storage and Retention

  • Runtime logs and activity entries may be retained for operations and debugging.
  • Retention and persistence depend on deployment configuration.
  • Do not submit secrets, personal data, or regulated data unless your deployment is configured for that use.

Third-Party Services

Depending on configuration, processing may involve Hugging Face infrastructure and optional third-party model/tool providers.
