What This Service Does

Champion Council is a model orchestration runtime and demo interface. It processes API requests, tool calls, and workflow operations in order to return results to users and connected clients.

Data We Process

  • Request metadata such as timestamps, tool names, and operation durations.
  • Arguments you send to endpoints such as /api/tool/{tool_name}.
  • Tool output and error payloads needed to return responses and show activity history.

How Data Is Used

  • To execute requested operations and return results.
  • To render operational telemetry in the Activity feed.
  • To diagnose runtime errors and improve reliability.

Storage and Retention

  • This public demo may retain short-term operational logs and recent activity events.
  • Persistence behavior depends on runtime configuration and enabled integrations.
  • Do not submit secrets, personal data, or regulated data to this public demo instance.

Third-Party Services

This service is hosted on Hugging Face Spaces. Processing may involve Hugging Face infrastructure and optional third-party model/tool providers enabled by the operator.

Public Demo Notice

This Space is intended for demonstration and testing. For private or production use, duplicate the Space and run under your own account and secrets.
