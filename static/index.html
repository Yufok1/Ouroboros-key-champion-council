Fork This Space in Minutes

Want your own copy without setup pain? Duplicate the template Space and add one secret.

  1. Open tostido/Champion_Council.
  2. Click the top-right menu (three dots) and choose Duplicate this Space.
  3. Choose your owner/name/hardware, then create the duplicate.
  4. In your new Space: Settings -> Secrets, add HF_TOKEN, then Factory Rebuild.

Important: Hugging Face copies public variables when duplicating, but private secrets are not copied.

Required secret and recommended variables

HF_TOKEN=hf_xxx
SPACE_AUTHOR_NAME=your_hf_username
SPACE_ID=your_space_name

Optional: keep your fork synced with upstream

git remote add upstream https://huggingface.co/spaces/tostido/Champion_Council
git fetch upstream
git merge upstream/main
git push origin main
